MAP BASE EXPORTER - HOW TO USE IT
Version 1.3.0
=================================

WHAT THIS APP DOES
------------------
It shows a full-colour street map on screen. You mark one or more
rectangular areas ("regions"), and it exports each one as a high-resolution
image (PNG or PDF) sized for print. You then bring those images into your
stitching app.


STARTING THE APP
----------------
1. Open a terminal in the "map-base-exporter" folder.
2. First time only: type  npm install  and press Enter. Wait for it to finish.
3. Type  npm run dev  and press Enter.
4. Open the web link it prints (usually http://localhost:5173) in your browser.

To stop the app, go back to the terminal and press Ctrl + C.


THE SCREEN
----------
- The map fills the screen.
- A control panel sits on the right. The top toolbar runs across the top.
- "Map only" in the toolbar hides everything for a clean view. Click it again
  to bring the controls back.


STEP 1 - MOVE THE MAP TO YOUR AREA
----------------------------------
- Drag the map to pan. Scroll to zoom in and out.
- Get your general area roughly centred before drawing regions.


STEP 2 - DRAW YOUR REGIONS
--------------------------
1. Click "Draw new region" (in the panel or the toolbar).
2. Drag a rectangle on the map. Let go to drop it.
3. Stay in draw mode and drag again to add more regions - draw as many as
   you want.
4. Click the button again ("Drawing - click to stop") when you're done.

Each region you draw becomes its own export file later.


STEP 3 - ADJUST A REGION (MOVE OR RESIZE)
-----------------------------------------
You do this in the normal (edit) mode - that's when you're NOT drawing.

- MOVE:   Click and drag the middle of a region to slide it around.
- RESIZE: Click a region to select it (handles appear on its edges and
          corners), then drag a handle to change its size.
- DELETE: Select a region and press the Delete or Backspace key,
          OR click the X next to it in the region list.

Regions stay locked to the ground - pan or zoom and they stay on the same
streets.


STEP 4 - NAME YOUR REGIONS (OPTIONAL BUT HANDY)
-----------------------------------------------
In the panel's "Regions" list, type a name into each region's box
(e.g. "Downtown", "North End"). The name is added to that region's file name,
so the exported images are easy to tell apart in your stitching app.


STEP 5 - SET THE OUTPUT
-----------------------
In the panel:
- Street names: turn the overlay on/off and set the font size.
- Document:     page size (Letter / Tabloid / A3 / A4 / Custom),
                portrait or landscape, PNG or PDF, and 150 or 300 DPI.
- Detail:       keeps your region as the exact boundary but re-renders it at
                higher zoom for more streets and crisper lines. Higher = bigger
                file. It shows the output size in pixels as you change it.

These settings apply to ALL regions in the same export run.


STEP 6 - EXPORT
---------------
1. Click the Export button at the bottom. If you have several regions it will
   say e.g. "Export 3 PNGs".
2. The app renders each region one after another and downloads each as its
   own file. The panel shows progress ("Region 2 of 3...").
3. THE FIRST TIME, your browser may ask "Allow this site to download multiple
   files?" - click Allow. After that it won't ask again.

File names look like:
   basemap_01-downtown_tabloid_landscape_150dpi_<date-time>.png

If you didn't draw any regions, Export just saves whatever the map is
currently showing as a single file.


STEP 7 - STITCHING
------------------
Bring the exported images into your stitching app as usual. The numbered,
named file names make it easy to line up your drilled-down areas.


TIPS
----
- Draw a few regions, name them, then export once - faster than doing them
  one at a time.
- Use "Map only" for a clean look while you're positioning the map.
- Higher Detail and 300 DPI make sharper but much larger files. Start at
  150 DPI and bump it up only if you need more.
